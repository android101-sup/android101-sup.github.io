Step 0: Emulator

  • OS X
  • Windows
  • Linux
  1. Navigate to Tools > Android > AVD Manager > Create Virtual Device.
  2. Choose a device. For this activity, we'll be using a Nexus 5 device.
  3. Chose a system image. For this activity, we'll be using Marshmallow. For more information about Android versions, check out the documentation here.
  4. Click finish.
  5. Press play on the device you just created. Give it a few seconds to start up.
